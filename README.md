# DataExtractorFromJPG
Python script to extract x,y data from a plot in a jpg file
Use "python extract_and_scale.py --img aa2219Data_3.jpg --num 25 --out extracted_points.csv" to run this python file. Change the jpg file name and the csv file where you want to save the x-y data extracted from the plot in the jpg file.
The script is ready to work with, just keep the jpg file at the same location where you save the python file.
The scripts imports matplotlib.pyplot, csv and argparse libraries so ensure these are already installed in the python environment.
